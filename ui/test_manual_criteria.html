Usage Instructions:
  1. Copy the example JSON from above
  2. Go to the main Performance Review Tracker UI
  3. In Step 1, select "Enter Manually" for either Annual Review or Competency Criteria
  4. Paste the JSON into the textarea
  5. Click "Validate JSON" to check the format
  6. Click "Save Criteria" to save and use in your analysis
